# sanitize-git

Review all commits on the current branch and clean up any AI attribution.

## Objective

1. Identify all commits on this branch not on the parent branch (default: `main`)
2. Remove AI attribution from commit messages
3. Ensure author/committer is the configured git user

## Forbidden Patterns to Remove

- "Created by Claude"
- "Generated by AI"
- "Co-authored-by: Claude"
- "Co-authored-by:.*[Cc]laude.*"
- "Co-authored-by:.*[Aa]nthrop.*"
- Any AI/assistant attribution in commit messages or trailers

## Execution Steps

### 1. Identify commits to clean

```bash
git log main..HEAD --oneline
```

### 2. Interactive rebase to amend commits

```bash
git rebase -i main
```

Mark commits needing cleanup as `reword` or `edit`.

### 3. For each commit with AI attribution

```bash
git commit --amend
git commit --amend --reset-author
```

### 4. Force push after cleanup

```bash
git push origin HEAD --force-with-lease
```

## Verification

```bash
git log main..HEAD --format="%B" | grep -iE "(claude|anthropic|co-authored-by.*ai|created by.*ai)"
```

Should return empty.
